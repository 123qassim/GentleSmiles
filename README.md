### Run Locally

To run **GentleSmiles** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/codewithsadee/GentleSmiles.git
```

Windows:

```bash
git clone https://github.com/codewithsadee/GentleSmiles.git
```
